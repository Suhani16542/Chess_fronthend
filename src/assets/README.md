# Assets Directory

Place static assets here that are imported directly in code (not in `public/`).

## Subdirectory Guide

| Folder       | Contents                                      |
|-------------|----------------------------------------------|
| `icons/`    | SVG icon files used as React components       |
| `images/`   | Static images imported in components          |
| `fonts/`    | Self-hosted font files (if not using Google)  |

> Images that need to be served at a public URL (e.g. OG images) go in `/public/` instead.
